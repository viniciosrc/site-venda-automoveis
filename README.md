O projeto desenvolvido pelo grupo consiste na criação do Velocità, um site voltado para a venda de automóveis, incluindo carros e motocicletas, novos e usados. A proposta da plataforma é oferecer aos usuários um ambiente seguro, moderno e intuitivo para a compra e venda de veículos.

Tecnologias utilizadas: HTML5, CSS3, JavaScript,Biblioteca de ícones 'Font Awesome' Bootstrap, 'API mockada' -> com funções do JS

Para utilizar localmente. Entre no reposiótio do GitHub e instale as pastas ou se preferir pesquise no google 'https://velocita.rf.gd'
